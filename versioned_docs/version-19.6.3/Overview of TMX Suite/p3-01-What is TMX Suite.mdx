title: What is TMX?
---

**TMX** is effective translation management software developed by BCS. **TMX** automates and streamlines the translation process eliminating repetitive tasks, making translation process more efficient. File and content management is significantly simplified by **TMX** because it centralises all files and information in one location.

Effective translation is a difficult process that requires a number of steps. **TMX** helps you set up tasks and deadlines, create a project plan and budget, and manage the translation project till completion.
