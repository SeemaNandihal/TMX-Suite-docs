title: Who is it for?
---
| Roles                     | Access                                                                                                                                                                                                                                                                                         |
|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Project Management Office(PMO) | PMO can create users, assign projects to project managers,  set each stages's services and associated payments and have full access to the TMX system. Tasks and the source file are imported,  Tasks are assigned to translators, and translated files are downloaded, reviewed and approved. |
| Project Manager           | Create Assignee(only translators), develop Projects, assign a supervisor or assistant project manager, import source file, assign tasks to the  translators, download and check the translated file, check PM dashboards,  create and view bills.                                              |
| Assistant Project Manager | Create Assignee(translators), create projects, Import tasks and source file, assign tasks to the translators, download the translated file, view PM dashboards, generate bills.                                                                                                                |
| Team Leader (TL)              | The project's team leader has access to the work and can view it, view TL dashboard, each team member's source file, and each member's completed file.                                                                                                                                         |
| Translators               | Translators can view dashboard, view assigned tasks, view and download the task file and upload the translated file.                                                                                                                                                                           |
| Accountant                | The accountant has access to all the bills that the project managers have submitted, and he/she can correct and approve them.                                                                                                                                                                      |
