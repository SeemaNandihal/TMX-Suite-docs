The task details are filled in by the assigned project manager.
The project manager adds a task, chooses a book code, specifies the kind of work required, and then set the due date, status, and priorities. For the reference of the assignees, adds a source file.When the task is finished, the project manager uploads the finished file to the attachment icon on the assignee dashboard.
