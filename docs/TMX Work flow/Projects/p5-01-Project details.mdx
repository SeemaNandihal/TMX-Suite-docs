title: Project details
---

The projects are shown in list and detail views. The detail view provides an overview of the selected project's details, task and stage details. The list view displays a list of all the projects along with details like project name, project status, such as ongoing and finished projects.  It also provides details regarding the project manager, the project's language, and its status.
