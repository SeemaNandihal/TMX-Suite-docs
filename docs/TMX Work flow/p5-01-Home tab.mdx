The **home page** of **TMX** displays a variety of project-related graphs that automatically update as new information about the projects become available.
The information provided on the Home Page is:

- Project status
- Project information with regard to  language
- Tasks
- Assignees by state
- Assignees by language
- Cost and time analysis
